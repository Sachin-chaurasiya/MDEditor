## React Markdown Editor Component

### To get started

#### Clone the repo

```shell
git clone https://github.com/Sachin-chaurasiya/MDEditor.git
```

#### Install the dependencies

```shell
npm install
```

#### Start the development server

```shell
npm start
```

Now visit the [localhost:3000](http://localhost:3000)
